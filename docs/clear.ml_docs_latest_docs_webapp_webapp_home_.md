The **project dashboard** provides a summary of your most recent projects, reports, and tasks. Click a project, report
or task to quickly access it.

![Dashboard](https://clear.ml/docs/latest/assets/images/webapp_dashboard-0e6d77c65e147b171d89529a2a36af88.png#light-mode-only)![Dashboard](https://clear.ml/docs/latest/assets/images/webapp_dashboard_dark-18623e104f3c010409a7b7b7172a9449.png#dark-mode-only)

To access the [projects page](https://clear.ml/docs/latest/docs/webapp/webapp_projects_page), click the `View All` button next to the recent projects summary.
Similarly, to access the [reports page](https://clear.ml/docs/latest/docs/webapp/webapp_reports), click the `View All` button next to the recent reports summary.

To create a new project, click the `+ New Project` button. Similarly, to create a new report, click the `+ New Report`.

To access the [orchestration page](https://clear.ml/docs/latest/docs/webapp/webapp_workers_queues), click `Manage Workers and Queues` (Open Source)/`Orchestration Control Center`.
